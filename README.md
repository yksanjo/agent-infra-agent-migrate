## Quick Start


```go
import "github.com/yksanjo/agent-infra-agent-migrate"

func main() {
    instance := AgentMigrate.New(AgentMigrate.Config{})
    instance.Initialize()
    result := instance.Execute(input)
}
```


## API Reference

### `AgentMigrate`

Main class for agent migrate functionality.

#### Methods

- `initialize()` - Initialize the component
- `execute(input)` - Execute main logic  
- `configure(config)` - Update configuration
